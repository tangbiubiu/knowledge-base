## 项目初始化
1. 安装node和vue
2. 使用以下命令初始化项目
```bash
npm create vue@latest
# 假设项目名为vue-project
cd vue-project
npm install
```

## 项目结构
```
.
├── env.d.ts
├── eslint.config.ts
├── index.html                          # 整个项目的入口
├── node_modules/                       # 存放依赖文件
├── package-lock.json
├── package.json                        # 定义依赖项和项目启动脚步
├── public/                             # 存放静态资源
├── README.md
├── src/                                # 存放源码，主要在这个目录里开发
│   ├── App.vue                         # 项目的首加载项。所有组件的主组件
│   ├── assets
│   │   ├── base.css
│   │   ├── logo.svg
│   │   └── main.css
│   ├── components
│   │   ├── HelloWorld.vue
│   │   ├── icons/
│   └── main.ts                         # 引导整个应用。vite的入口脚本             
├── tsconfig.app.json
├── tsconfig.json
├── tsconfig.node.json
└── vite.config.ts
```
